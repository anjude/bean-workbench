# 模板合集

> 模板是骨架，不是枷锁。按内容自然调整，不要为了填模板制造段落、页数或文件。

## 默认文件组合

### 公众号文章

默认只需要：

- `公众号文章.md`（含选题区 + 正文）
- `发布检查清单.md`
- `复盘.md`

按需增加：

- `图文脚本.md`：需要拆成小红书图文
- `纯享正文.md`：需要跨平台复制版

### 小红书图文

默认需要：

- `图文脚本.md`
- `纯享正文.md`
- `发布检查清单.md`
- `复盘.md`

页数按内容自然决定，通常 4-8 页，不强行凑 6 页。

## 发布检查清单.md

```markdown
# 发布检查清单

## 基本信息

- 内容标题：
- 内容形式：
- 检查日期：
- 当前结论：

## 合规

- [ ] 无保本保收益
- [ ] 无具体收益率
- [ ] 无买入、卖出、上车、跟买建议
- [ ] 无开户或交易诱导
- [ ] 未把名人动作写成投资信号
- [ ] 已保留免责声明

## 文风

- [ ] 有自然入口
- [ ] 没有预设读者想法
- [ ] 没有解释笑点
- [ ] 没有主持人口吻转场
- [ ] 没有强行升华
- [ ] 每段有新增信息量
- [ ] 粗体数量适中

## 发布前确认

- [ ] 标题最终确定
- [ ] 事实源已核对
- [ ] 如使用截图或引用，来源清楚
- [ ] 留言引导自然
```
