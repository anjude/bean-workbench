# 目标管理

## 工作台长期目标

形成个人可持续进化的 Agent 工作台，通过 `skill`、知识库、任务记录和脚本提高业务仓代码与文档产出效率。

## 当前阶段目标

1. 建立稳定的工作台目录结构。
2. 建立全局可触发的 `hair` skill。
3. 建立轻量业务仓路由能力。
4. 建立知识库、自进化和任务登记机制。

## 验收标准

- Agent 能通过 `hair` 进入工作台上下文。
- `task/registry.md` 能反映热任务和路由提示。
- 工作台结构能通过 `make check`。
- 新增业务经验能落到知识库、skill 或脚本中的合适位置。
